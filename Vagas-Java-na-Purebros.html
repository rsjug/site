A Pure Bros uma integradora de SVA que atua nos mercados do Brasil, Itália e México, e tem como grande diferencial o foco em integração de empresas mobile com operadoras celulares.

Desenvolvedor Java Pleno - web (1 vaga) Necessária experiência em programação Java, Spring MVC e My SQL. Formação superior completa ou em andamento em Tecnologia ou áreas afins e inglês nível intermediário. Desejável experiência em desenvolvimento web utilizando JSP, Javascript, Prototype, JSTL, Jasper Report, JDBC e Tomcat.

Desenvolvedor Java Pleno - Backend (1 vaga) Necessária experiência em programação Java (backend), My SQL, Linux, Shell Script e programação concorrente e distribuída. Formação superior completa ou em andamento em tecnologia ou áreas afins e inglês nível intermediário. Desejável conhecimento em frameworks de desenvolvimento Java e protocolos de comunicação (HTTP, SMPP).
